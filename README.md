# Python CGPA Calculator 🎓

This is a beginner-level Python program that calculates the CGPA (out of 4.0) for a semester based on user input.

## 📌 Features

- Accepts semester number (1–8 only)
- Accepts up to 6 subjects
- Takes grade points (0–4) for each subject
- Validates user input
- Calculates and displays CGPA

## 🛠 How It Works

1. User enters semester number.
2. Program checks if it is between 1 and 8.
3. User enters number of subjects (maximum 6).
4. Program asks for grade points for each subject.
5. CGPA is calculated using:

   CGPA = Total Grade Points / Number of Subjects

## ▶️ How to Run

Make sure Python is installed on your system. The code is in the .py file.
